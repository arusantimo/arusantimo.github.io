# 대체거래소(ATS) 시가베팅 실행 루틴

> 추천 엔진: [`tools/open-bet-analyze/`](../tools/open-bet-analyze/)  
> 분석 지침: [`morning-open-bet-routine.md`](morning-open-bet-routine.md)

## 하루 흐름 (KST)

| 시각 | 할 일 |
|------|--------|
| **07:30~07:40** | `run_open_bet.py --phase pre_ats` 실행 → TOP3·지정가 출력 |
| **07:40~08:00** | **직접 종목 검토·판단·주문 준비** (20분) |
| **08:00** | 대체거래소(ATS)에서 추천 종목 **지정가 매수** |
| **08:00~08:30** | 목표가 도달 시 분할 매도 · 손절 라인 이탈 시 전량 매도 |
| **08:30** | ATS 구간 **전량 청산** 목표 (당일 단타 종료) |
| **08:55 전** | 09:00용 **잔량 지정가 매도** 주문 예약 (못 판 수량만) |
| **09:00** | 정규장 개장 → 예약 주문 자동 접수·체결 |

## 08:00~08:30 ATS 매매

- **매수**: 엔진 `buyOrder.limitPrice` (시간외·예상 시가 참고)
- **1차 익절 (08:15 전후)**: 보유 수량 **50%**, `tp1Price` (+2% 근처)
- **전량 청산 (08:30 전)**: 남은 수량 **100%** 매도
- **손절**: `stopLoss.limitPrice` 이탈 시 즉시 전량

## 09:00 잔량 시스템 매도

08:30까지 팔지 못한 수량이 있으면:

1. HTS/MTS에서 **09:00 정규장 지정가 매도** 예약
2. 가격: 대시보드 **`krxLiquidation.limitPrice`** (예상 시초가 또는 1차 익절가 중 보수적 값)
3. 수량: ATS에서 **미청산 잔량 전부**

엔진 JSON 필드: `krxLiquidationOrders[]`

## 체크리스트

```
□ 07:40 — pre_ats 배치 완료·TOP3·지정가 확인
□ 07:40~08:00 — 종목 재검토·진입 여부 최종 판단
□ 08:00 — ATS 매수 지정가 주문
□ 08:15 — 1차 익절 50% 체크
□ 08:30 — 잔량 0 확인 (ATS)
□ 08:55 — 09:00 잔량 매도 예약 입력
□ 09:00 — 예약 체결·잔량 0 재확인
```
