## 1. 공통 사전 필터

아래 조건 불충족 시 즉시 제외합니다.

- 당일 국내 주식 총 거래대금 **30위 이내** 종목만 분석 대상
- 실적발표 **D-1**, 분할/합병/배당락 **D-3** 이내 종목 제외
- 감리·경고·관리종목, 전일 거래정지·단일가 매매 종목 제외
- 공시 위반 이력 **1개월 이내** 종목 제외

> **전략 ③ 추가 필터**: 시가총액 **30조원 이상** 대형주 한정. 실적발표 D-2, 분할/합병/배당락 D-5 이내 제외 (보수적 적용). 자세한 내용은 §9 참조.

## 2. 이동평균선 계층 구조

**핵심 원칙: 보유 기간과 이동평균 기간을 매칭한다.**

### 이평선별 역할 정의

| 이평선 | 역할 | 의미 |
|---|---|---|
| 60MA | **시장 참가 여부 결정** | 장기 추세 필터 |
| 20MA | **진입 방향 + 스윙 전환 기준선** | 중기 추세 본체 |
| 5MA | **진입 타이밍 + 청산 트리거** | 단기 모멘텀 |

### 전략 × 이평선 매트릭스

| 전략 / 단계 | 1차 (핵심 신호) | 2차 (방향 필터) | 3차 (참고) |
|---|---|---|---|
| 눌림목 종가베팅 | 5MA + 20MA (정배열·이격) | 60MA (위치 확인) | — |
| 수급매집형 종가베팅 | 5MA (돌파 자리) | 20MA (추세 지지) | 60MA |
| **주도주 급락 반등 (신규)** | **60MA (베이스 사수)** | **20MA (수익 목표)** | **5MA (단기 추세 회복)** |
| 시가베팅 (오전 지침) | 5MA + 분봉 5MA | 20MA (방향) | — |
| 종베 → 스윙 전환 운용 | 20MA (유지 기준선) | 60MA (이탈 시 강제 청산) | 5MA (트레일링) |

> **전략 ③의 이평선 활용이 역방향인 이유**: 다른 전략은 "5MA 위에서 매수 → 60MA 이탈 시 손절"의 추세 추종 구조이지만, 본 전략은 **"60MA 사수 확인 후 진입 → 5MA·20MA 회복을 수익으로 환수"** 하는 역추세 구조입니다.

## 3. 시장 레짐 체크 — 4분류

매일 분석 전 가장 먼저 확인합니다. **위에서 아래로 순차 판정**하여 처음 일치하는 레짐으로 확정합니다.

### 판정 기준

| 우선순위 | 레짐 | 판정 조건 |
|---|---|---|
| 1 | 약세장 ⛔ | KOSPI 60MA 하락 **OR** VKOSPI > 30 |
| 2 | 강세장 ✅ | 60MA 우상향 **AND** 20MA 우상향 **AND** VKOSPI < 20 |
| 3 | 순환매장 🔄 | 60MA 횡보 **AND** 신고가 종목 ≥ 30개 **AND** 업종 1위 직전 3일 대비 변경 **AND** 거래대금 20일 평균 ±15% 유지 |
| 4 | 박스권 ⚠️ | 그 외 (지수 정체 + 내부 침체) |

### 순환매장 vs 박스권 — 핵심 차이

| 지표 | 박스권 | 순환매장 |
|---|---|---|
| 신고가 종목 수 | < 10개 | ≥ 30개 |
| 거래대금 | 평균 대비 감소 | 평균 유지 또는 증가 |
| 업종 리더십 | 정체 | 매일 교체 |
| 등락주선(ADL) | 횡보·하락 | 우상향 |

### 데이터 확인 위치

- VKOSPI: 네이버 증권 → 국내증시 → 변동성지수
- 신고가 종목 수: 네이버 증권 → 국내증시 → 신고가/신저가
- 업종 등락률: 네이버 증권 → 국내증시 → 업종별 시세

## 5. 점수 등급 체계

### 등급 기준 (추세 추종 전략 — 전략 ①·②)

| 등급 | 점수 | 의미 |
|---|---|---|
| **S — 강력매수** | 9.0 ~ 10점 | 레짐 무관 진입 가능 |
| **A — 매수추천** | 7.5 ~ 8.9점 | 강세장·순환매장·박스권 진입 가능 |
| **B — 관심후보** | 6.0 ~ 7.4점 | 매매 단계 노출, 진입 보류, 익일 재평가 |
| **C — 제외** | 6.0점 미만 | 출력 목록에서 제외 |

### 등급 기준 (역추세 전략 — 전략 ③, 0.5점씩 상향)

| 등급 | 점수 | 의미 |
|---|---|---|
| **S — 강력 반등** | 8.5 ~ 10점 | 최우선 진입 |
| **A — 반등 시도** | 7.0 ~ 8.4점 | 진입 가능 |
| **B — 관심 후보** | 5.5 ~ 6.9점 | 익일 재평가 (당일 진입 금지) |
| **C — 제외** | 5.5점 미만 | 출력 제외 |

> **등급 경계가 다른 이유**: 역추세는 신호 신뢰도가 더 높아야 매매 가능합니다. 추세 추종은 "방향이 맞으면 시간이 보상"하지만, 역추세는 "신호가 약하면 칼을 받는" 구조입니다.

### 레짐 × 등급 매매 가능 여부 (추세 추종)

|  | S (9.0+) | A (7.5~8.9) | B (6.0~7.4) |
|---|---|---|---|
| 강세장 | ✅ 100% 진입 | ✅ 100% 진입 | 👀 모니터링 |
| 순환매장 | ✅ 80% 진입 | ✅ 70% 진입 | 👀 모니터링 |
| 박스권 | ✅ 70% 진입 | ✅ 70% 진입 | 👀 모니터링 |
| 약세장 | ✅ 50% 진입 | ❌ 매매금지 | 👀 모니터링 |

### VKOSPI 점수 보정

#### 추세 추종 전략 (①·②)

| VKOSPI | 점수 보정 |
|---|---|
| < 20 | 보정 없음 |
| 20 ~ 30 | 원점수 × 0.9 |
| > 30 | 원점수 × 0.8 |

#### 역추세 전략 (③) — 보정 방향이 반대

| VKOSPI | 점수 보정 |
|---|---|
| < 20 | 원점수 × 0.8 (변동성 낮은 환경에서는 신호 진정성 약함) |
| 20 ~ 30 | **원점수 × 1.0** (정상 작동 구간) |
| > 30 | 원점수 × 0.9 |

## 9. 전략 ③ 주도주 급락 반등 매매 (신규 v3.5)

> **모듈 분리 원칙**: 본 전략은 **역추세(Counter-Trend) 단기 매매**입니다. 기존 4개 전략(눌림목·수급매집·시가베팅·스윙 전환)은 모두 추세 추종 시스템이므로, 본 전략의 신호가 기존 전략을 오염시키지 않도록 **별도 트랙으로 격리 운용**합니다.

### 9.1 전략 정의 및 위치

| 항목 | 내용 |
|---|---|
| 분류 | **역추세(Counter-Trend) 단기 매매** |
| 보유 기간 | 최대 **3거래일** (스윙 전환 절대 금지) |
| 신호 발생 빈도 | 월 2~5회 (희소 이벤트성) |

| 우선순위 | 기존 4개 전략과 **병렬**, 매트릭스 상 별도 트랙 |

**철학**: 단순 "많이 빠졌다"는 가격 신호가 아니라, **매도 클라이맥스 직후 매수세 진입 신호**를 포착합니다. 잘못 잡으면 칼을 받는 매매이므로 Gate를 매우 좁게 설정하고, 진입 후에는 빠른 회수·빠른 손절을 원칙으로 합니다.

### 9.9 스윙 전환 절대 금지 원칙

본 전략의 모든 포지션은 **D+3 종가에 강제 청산**합니다. V1~V5 자발적 전환, I1~I5 비자발적 전환 양쪽 모두 적용 불가입니다.

추세가 회복되어 추가 보유 매력이 발생한 경우, **본 포지션은 청산하고, 별도로 눌림목 종가베팅 신규 종목으로 재발굴**하시는 것이 시스템 정합성에 부합합니다. 이는 두 가지 이유 때문입니다.

1. 역추세 신호와 추세 추종 신호의 운용 규칙이 본질적으로 다름
2. 동일 종목을 다른 시스템으로 재진입할 경우 진입 가격과 손절선이 새롭게 설정되어야 함

## 10. 시가베팅 시그널 판단 (종가 분석 시점)

> **이 섹션의 역할**: 종가 분석 시점에서 "시가베팅 전략이 활성화되는 레짐인지" 판단하고, Gate 조건을 1차 통과하는 종목을 **시그널 후보**로 플래그합니다. 실제 종목 선정·진입가·매도 전략은 **익일 오전 시가베팅 전용 지침**에서 수행합니다.

### Step 1: 레짐 활성 여부 판단

```
현재 레짐이 강세장(✅) 또는 순환매장(🔄)인가?
  → YES: 시가베팅 시그널 스캔 진행
  → NO: "시가베팅 비활성" 출력 후 종료
```

### Step 2: 시그널 후보 1차 스크리닝 (Gate 조건 사전 확인)

거래대금 30위 이내 종목 중 아래 3개 Gate를 전부 충족하는 종목만 플래그합니다.

| # | 조건 | 확인 시점 |
|---|---|---|
| G1 | 당일 종가 > 5MA + 5MA 우상향 | 장마감 후 |
| G2 | 당일 거래량 ≥ 5일 평균 150% | 장마감 후 |
| G3 | 당일 꽉찬 양봉 (종가 ≥ 고가 × 0.95) | 장마감 후 |

### Step 3: 출력 (종가 분석 보고서 내)

```markdown
### ⏰ 시가베팅 시그널 (익일 오전 재확인 필요)

레짐 활성: ✅ [강세장/순환매장]

시그널 후보:
1. 종목명 (코드) — G1✅ G2✅ G3✅ | 종가 OOO원 (+O.O%)
2. 종목명 (코드) — G1✅ G2✅ G3✅ | 종가 OOO원 (+O.O%)
3. 종목명 (코드) — G1✅ G2✅ G3✅ | 종가 OOO원 (+O.O%)

⚠️ 최종 진입 판단은 익일 08:00~08:50 오전 지침에서 수행합니다.
   (시간외 단일가·갭업 범위·추가 Gate G4/G5 확인 필요)
```

### 판단 불가 시

Gate 조건 충족 종목이 0개인 경우:
```
시가베팅 시그널: 해당 없음 (Gate 통과 종목 부재)
```

## 11. 운용 단계: 종베 → 스윙 전환 규칙

> 본 섹션은 **신규 진입 전략이 아닙니다**. 종가베팅(눌림목·수급매집형) 진입 후, **D+1 종가 마감 시점**에 매도 또는 스윙 전환을 결정하는 운용 게이트입니다.
>
> **전략 ③ 주도주 급락 반등 매매는 본 섹션 적용 대상이 아닙니다.** 전략 ③은 D+3 강제 청산으로 별도 운용됩니다.

### 평가 시점

**종베 진입 후 1거래일차 종가 마감 직후** (당일 매도 단계 종료 시점)

### 평가 분기 흐름

```
종베 진입 → D+1 매도 단계 수행
              │
              ├─ 1차/2차 익절 완료 + 잔량 보유 중
              │           │
              │           ↓
              │     ┌─ 자발적 전환 평가 (V1~V5)
              │     │     모두 충족 → ✅ 스윙 전환
              │     └─ 1개라도 미충족 → 잔량 전량 청산
              │
              ├─ 매도 라인 미도달 (갭다운 등)
              │           │
              │           ↓
              │     ┌─ 비자발적 전환 평가 (I1~I5)
              │     │     모두 충족 → ⚠️ 조건부 스윙 전환
              │     └─ 1개라도 미충족 → 즉시 손절
              │
              └─ 거부 조건 (R1~R4) 1개라도 충족
                          │
                          ↓
                    ❌ 무조건 손절 (예외 없음)
```

### 자발적 전환 조건 (V — Voluntary)

**모두 충족 시 스윙 전환 ✅**

| # | 조건 |
|---|---|
| V1 | 1차 익절 완료 (보유량의 30~50% 이상 수익 실현 완료) |
| V2 | 잔량 수익률 ≥ +5% |
| V3 | 종가 > 5MA + 5MA 우상향 유지 |
| V4 | 당일 거래량 ≥ 5일 평균 (수급 지속) |
| V5 | 외국인 OR 기관 순매수 유지 |

### 비자발적 전환 조건 (I — Involuntary)

**전부 충족 시에만 조건부 허용 ⚠️**

| # | 조건 |
|---|---|
| I1 | 시초가 갭다운 -1% 이내로 익절 라인 미달 |
| I2 | 종가 > **20MA** (반드시 20MA 위) |
| I3 | 종가 ≥ 진입가 -1% (실손실 1% 미만) |
| I4 | 외국인 + 기관 합산 순매도 아님 |
| I5 | 거래량 ≥ 5일 평균의 70% (관심 이탈 아님) |

> **하나라도 미충족 시 → 매도 (스윙 전환 금지)**
>
> 비자발적 전환은 **손절 회피의 합리화를 차단하기 위한 엄격한 게이트**입니다. 의심스러우면 매도가 원칙입니다.

### 전환 거부 조건 (R — Reject)

**1개라도 충족 시 즉시 손절, 예외 없음 ❌**

| # | 조건 |
|---|---|
| R1 | 20MA 이탈 종가 |
| R2 | 거래량 5일 평균의 50% 이하 (관심 이탈) |
| R3 | 외국인 + 기관 동시 순매도 |
| R4 | 종가 < 진입가 -3% (눌림목) / -4% (수급매집) |

### 전환 후 운용 규칙

**3중 강제 청산 트리거** (우선순위 1)

| 트리거 | 조건 | 비고 |
|---|---|---|
| 시간 | 진입일로부터 **15거래일** 경과 | 무조건 1순위 (순환매장은 5거래일) |
| 가격 | **20MA 이탈 종가** | 즉시 청산 |
| 추세 | **60MA 이탈** | 무조건 청산 (예외 없음) |

**일반 매도 단계**

| 단계 | 조건 | 행동 | 목표 |
|---|---|---|---|
| 1차 트레일링 익절 | 첫 음봉 출현 OR 추가 +3% | 30% 정리 | — |
| 2차 트레일링 익절 | 5MA 이탈 | 40% 추가 정리 | — |
| 3차 마감 청산 | 박스 상단 도달 OR 강제 트리거 발동 | 잔량 전량 | — |
| 손절 | -5% (전환일 종가 대비) OR 20MA 이탈 | 전량 | — |

        └─ 거래대금 상위 30종목 추출
        └─ 공통 사전 필터 적용
        └─ 전략별 Gate 조건 확인 (전략 ①·②)

Step 4-a. 주도주 급락 반등 후보 발굴 (역추세 트랙, Step 4 병렬)

          └─ G5-b/c 종목 → 익일 시가 분할 진입 플래그

Step 9. 뉴스 인사이트 수집

Step 10. 결과 출력
         └─ D+1 평가 + 종가 진입 + 급락 반등 후보 + 시가 시그널 + 뉴스

## 부록 C: 설계 철학 메모

### 시가베팅 분리 근거

시가베팅의 최종 진입 결정은 장전 시간외 단일가(08:30~)와 갭업 범위(09:00)에 의존합니다. 전일 종가 시점에서 종목을 확정하면 "갭업 +4% 이상 진입 보류" 규칙이 반영될 수 없고, 시간외 뉴스·공시(G5)를 놓칩니다. 따라서 종가 분석은 "이 종목이 내일 시가베팅 후보가 될 수 있는가?"까지만 판단하고, 실제 진입은 오전 루틴에 위임합니다.

### 전략 ③ (주도주 급락 반등 매매) 설계 근거

**왜 별도 트랙으로 격리하는가**

기존 4개 전략은 모두 **추세 추종(Trend-Following)** 시스템입니다. "5MA 위에서 매수 → 추세 이탈 시 손절"의 일관된 구조를 공유합니다. 반면 본 전략은 **"60MA 사수 확인 후 매수 → 5MA·20MA 회복을 수익으로 환수"** 하는 역추세 구조이므로 본질적으로 다른 시스템입니다.

만약 한 시스템 안에 두 가치관을 섞으면, 같은 종목·같은 시점에 한 전략은 "매수"를 외치고 다른 전략은 "관망"을 외치는 신호 충돌이 발생합니다. 격리 운용을 통해 각 시스템의 가설이 독립적으로 검증·개선될 수 있도록 했습니다.

**왜 시가총액 30조원 이상 대형주 한정인가**

중소형주의 -10% 급락은 종종 추세 전환의 1차 신호입니다. 반면 대형주(시총 30조원 이상)는 단일 종목의 펀더멘털보다 시장 전체의 자금 흐름이 주가를 결정하므로, 급락 후 반등 확률이 통계적으로 높습니다. 또한 거래대금이 풍부하여 분할 진입·청산이 용이합니다.

**왜 D+3 강제 청산인가**

역추세 매매의 본질은 "단기 평균회귀"입니다. 3거래일 내 반등하지 않으면 그 시나리오는 기각된 것으로 간주합니다. 추세가 회복되어 추가 보유 매력이 발생하면, 같은 종목을 별도 전략(눌림목 또는 수급매집)으로 재발굴하여 진입 가격과 손절선을 새롭게 설정합니다. 이는 시스템 정합성을 보장하고, 손실 회피 편향을 차단하는 장치입니다.

**왜 포지션 크기를 50%로 제한하는가**

역추세 매매는 추세 추종 대비 승률은 비슷하나 손절폭이 빠르게 발생하는 특성이 있습니다. 동일 자금 노출 시 변동성이 크므로, 단위 포지션을 줄여 전체 포트폴리오의 위험 균형을 맞춥니다.

### 15.3 블록 구성 순서

페이지 내 블록은 아래 순서를 고정합니다. **§14 출력 형식의 섹션 순서와 동일**하며, 해당 없는 블록은 **"해당 없음"** 한 줄로 기재하고 생략하지 않습니다 (이력 일관성 + 외부 파싱 도구 호환 보장).
