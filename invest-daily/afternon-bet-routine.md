# 종가베팅 시장 분석 루틴 v3.4

종가베팅(눌림목·수급매집형) · 시가베팅 시그널 판단 · 종베→스윙 전환 운용을 위한 일일 시장 분석 지침입니다.

> **v3.3 → v3.4 핵심 변경 사항**
>
> **시가베팅 처리 방식 변경**: 종가 분석 시점에서 시가베팅을 "종목 선정·채점"까지 수행하지 않고, **레짐 해당 여부 판단 + 시그널 후보 플래그**만 수행합니다.
>
> - 종가 분석 출력 시: "시가베팅 레짐 활성 여부" + "시그널 감지 종목 리스트(간단)" 제공
> - 실제 시가베팅 종목 선정·진입가·매도 전략: **익일 오전(08:00~08:50) 전용 지침**에서 별도 수행
>
> 이 변경은 시가베팅이 레짐(순환매장·강세장)에서만 활성화되며, 최종 진입 판단은 장전 시간외 단일가·갭업 범위 확인 후에야 유효하다는 실전 원칙을 반영합니다.

## 1. 공통 사전 필터

아래 조건 불충족 시 즉시 제외합니다.

- 당일 국내 주식 총 거래대금 **30위 이내** 종목만 분석 대상
- 실적발표 **D-1**, 분할/합병/배당락 **D-3** 이내 종목 제외
- 감리·경고·관리종목, 전일 거래정지·단일가 매매 종목 제외
- 공시 위반 이력 **1개월 이내** 종목 제외

## 2. 이동평균선 계층 구조

**핵심 원칙: 보유 기간과 이동평균 기간을 매칭한다.**

### 이평선별 역할 정의

| 이평선 | 역할 | 의미 |
|---|---|---|
| 60MA | **시장 참가 여부 결정** | 장기 추세 필터 |
| 20MA | **진입 방향 + 스윙 전환 기준선** | 중기 추세 본체 |
| 5MA | **진입 타이밍 + 청산 트리거** | 단기 모멘텀 |

### 전략 × 이평선 매트릭스

| 전략 / 단계 | 1차 (핵심 신호) | 2차 (방향 필터) | 3차 (참고) |
|---|---|---|---|
| 눌림목 종가베팅 | 5MA + 20MA (정배열·이격) | 60MA (위치 확인) | — |
| 수급매집형 종가베팅 | 5MA (돌파 자리) | 20MA (추세 지지) | 60MA |
| 시가베팅 (오전 지침) | 5MA + 분봉 5MA | 20MA (방향) | — |
| **종베 → 스윙 전환 운용** | **20MA (유지 기준선)** | **60MA (이탈 시 강제 청산)** | 5MA (트레일링) |

## 3. 시장 레짐 체크 — 4분류

매일 분석 전 가장 먼저 확인합니다. **위에서 아래로 순차 판정**하여 처음 일치하는 레짐으로 확정합니다.

### 판정 기준

| 우선순위 | 레짐 | 판정 조건 |
|---|---|---|
| 1 | 약세장 ⛔ | KOSPI 60MA 하락 **OR** VKOSPI > 30 |
| 2 | 강세장 ✅ | 60MA 우상향 **AND** 20MA 우상향 **AND** VKOSPI < 20 |
| 3 | 순환매장 🔄 | 60MA 횡보 **AND** 신고가 종목 ≥ 30개 **AND** 업종 1위 직전 3일 대비 변경 **AND** 거래대금 20일 평균 ±15% 유지 |
| 4 | 박스권 ⚠️ | 그 외 (지수 정체 + 내부 침체) |

### 순환매장 vs 박스권 — 핵심 차이

| 지표 | 박스권 | 순환매장 |
|---|---|---|
| 신고가 종목 수 | < 10개 | ≥ 30개 |
| 거래대금 | 평균 대비 감소 | 평균 유지 또는 증가 |
| 업종 리더십 | 정체 | 매일 교체 |
| 등락주선(ADL) | 횡보·하락 | 우상향 |

### 데이터 확인 위치

- VKOSPI: 네이버 증권 → 국내증시 → 변동성지수
- 신고가 종목 수: 네이버 증권 → 국내증시 → 신고가/신저가
- 업종 등락률: 네이버 증권 → 국내증시 → 업종별 시세

## 9. 시가베팅 시그널 판단 (종가 분석 시점)

> **이 섹션의 역할**: 종가 분석 시점에서 "시가베팅 전략이 활성화되는 레짐인지" 판단하고, Gate 조건을 1차 통과하는 종목을 **시그널 후보**로 플래그합니다. 실제 종목 선정·진입가·매도 전략은 **익일 오전 시가베팅 전용 지침**에서 수행합니다.

### Step 1: 레짐 활성 여부 판단

```
현재 레짐이 강세장(✅) 또는 순환매장(🔄)인가?
  → YES: 시가베팅 시그널 스캔 진행
  → NO: "시가베팅 비활성" 출력 후 종료
```

### Step 2: 시그널 후보 1차 스크리닝 (Gate 조건 사전 확인)

거래대금 30위 이내 종목 중 아래 3개 Gate를 전부 충족하는 종목만 플래그합니다.

| # | 조건 | 확인 시점 |
|---|---|---|
| G1 | 당일 종가 > 5MA + 5MA 우상향 | 장마감 후 |
| G2 | 당일 거래량 ≥ 5일 평균 150% | 장마감 후 |
| G3 | 당일 꽉찬 양봉 (종가 ≥ 고가 × 0.95) | 장마감 후 |

### Step 3: 출력 (종가 분석 보고서 내)


## 10. 운용 단계: 종베 → 스윙 전환 규칙

> 본 섹션은 **신규 진입 전략이 아닙니다**. 종가베팅(눌림목·수급매집형) 진입 후, **D+1 종가 마감 시점**에 매도 또는 스윙 전환을 결정하는 운용 게이트입니다.

### 평가 시점

**종베 진입 후 1거래일차 종가 마감 직후** (당일 매도 단계 종료 시점)

### 평가 분기 흐름

```
종베 진입 → D+1 매도 단계 수행
              │
              ├─ 1차/2차 익절 완료 + 잔량 보유 중
              │           │
              │           ↓
              │     ┌─ 자발적 전환 평가 (V1~V5)
              │     │     모두 충족 → ✅ 스윙 전환
              │     └─ 1개라도 미충족 → 잔량 전량 청산
              │
              ├─ 매도 라인 미도달 (갭다운 등)
              │           │
              │           ↓
              │     ┌─ 비자발적 전환 평가 (I1~I5)
              │     │     모두 충족 → ⚠️ 조건부 스윙 전환
              │     └─ 1개라도 미충족 → 즉시 손절
              │
              └─ 거부 조건 (R1~R4) 1개라도 충족
                          │
                          ↓
                    ❌ 무조건 손절 (예외 없음)
```

### 자발적 전환 조건 (V — Voluntary)

**모두 충족 시 스윙 전환 ✅**

| # | 조건 |
|---|---|
| V1 | 1차 익절 완료 (보유량의 30~50% 이상 수익 실현 완료) |
| V2 | 잔량 수익률 ≥ +5% |
| V3 | 종가 > 5MA + 5MA 우상향 유지 |
| V4 | 당일 거래량 ≥ 5일 평균 (수급 지속) |
| V5 | 외국인 OR 기관 순매수 유지 |

### 비자발적 전환 조건 (I — Involuntary)

**전부 충족 시에만 조건부 허용 ⚠️**

| # | 조건 |
|---|---|
| I1 | 시초가 갭다운 -1% 이내로 익절 라인 미달 |
| I2 | 종가 > **20MA** (반드시 20MA 위) |
| I3 | 종가 ≥ 진입가 -1% (실손실 1% 미만) |
| I4 | 외국인 + 기관 합산 순매도 아님 |
| I5 | 거래량 ≥ 5일 평균의 70% (관심 이탈 아님) |

> **하나라도 미충족 시 → 매도 (스윙 전환 금지)**
>
> 비자발적 전환은 **손절 회피의 합리화를 차단하기 위한 엄격한 게이트**입니다. 의심스러우면 매도가 원칙입니다.

### 전환 거부 조건 (R — Reject)

**1개라도 충족 시 즉시 손절, 예외 없음 ❌**

| # | 조건 |
|---|---|
| R1 | 20MA 이탈 종가 |
| R2 | 거래량 5일 평균의 50% 이하 (관심 이탈) |
| R3 | 외국인 + 기관 동시 순매도 |
| R4 | 종가 < 진입가 -3% (눌림목) / -4% (수급매집) |

### 전환 후 운용 규칙

**3중 강제 청산 트리거** (우선순위 1)

| 트리거 | 조건 | 비고 |
|---|---|---|
| 시간 | 진입일로부터 **15거래일** 경과 | 무조건 1순위 (순환매장은 5거래일) |
| 가격 | **20MA 이탈 종가** | 즉시 청산 |
| 추세 | **60MA 이탈** | 무조건 청산 (예외 없음) |

**일반 매도 단계**

| 단계 | 조건 | 행동 | 목표 |
|---|---|---|---|
| 1차 트레일링 익절 | 첫 음봉 출현 OR 추가 +3% | 30% 정리 | — |
| 2차 트레일링 익절 | 5MA 이탈 | 40% 추가 정리 | — |
| 3차 마감 청산 | 박스 상단 도달 OR 강제 트리거 발동 | 잔량 전량 | — |
| 손절 | -5% (전환일 종가 대비) OR 20MA 이탈 | 전량 | — |

## 12. 분석 실행 순서

```
Step 1. 시장 레짐 확인 (4분류)
        └─ KOSPI 60MA·20MA + VKOSPI + 신고가 종목 수
        └─ 레짐 확정

Step 2. 레짐 × 전략 우선순위 매트릭스 적용
        └─ 활성 종가 전략과 보류 전략 분리
        └─ 시가베팅 레짐 활성 여부 결정
        └─ 스윙 전환 활성도 결정

Step 3. 전일 보유 종목 D+1 평가 (스윙 전환 게이트)
        └─ 전일 종베 진입 종목 전수 평가
        └─ 거부 조건(R1~R4) 우선 체크 → 해당 시 즉시 손절
        └─ 자발적(V1~V5) 또는 비자발적(I1~I5) 전환 판정

Step 4. 신규 종가 진입 종목 발굴
        └─ 거래대금 상위 30종목 추출
        └─ 공통 사전 필터 적용
        └─ 전략별 Gate 조건 확인

Step 5. 채점 조건 적용
        └─ 수급·위치·캔들 조건 점수 산출
        └─ 체결강도(토스)·호가잔량(토스) 확인
        └─ VKOSPI 보정 후 최종 점수

Step 6. 등급 분류 및 정렬
        └─ C등급(6.0점 미만) 제외
        └─ 전략별 상위 종목 선정

Step 7. 매도 전략 산출 (스윙 전환 라인 포함)
        └─ 레짐 × 전략 × 등급별 단계 목표가 계산

Step 8. 시가베팅 시그널 판단
        └─ 레짐 활성 여부 확인
        └─ Gate G1~G3 1차 스크리닝
        └─ 시그널 후보 플래그 (최대 3종목)

Step 9. 뉴스 인사이트 수집

Step 10. 결과 출력
         └─ D+1 평가 + 종가 진입 + 시가 시그널 + 뉴스
```
