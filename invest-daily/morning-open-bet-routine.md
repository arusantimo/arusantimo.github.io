# 오전 시가베팅 분석 루틴 v1.0

장 시작 전(08:00~08:50) 시가베팅 종목 최종 선정 및 진입 전략 수립을 위한 전용 지침입니다.

> **전제 조건**: 이 루틴은 전일 종가 분석(v3.4)에서 "시가베팅 레짐 활성" 판정 + "시그널 후보 플래그"가 존재할 때만 실행됩니다.
>
> 전일 종가 분석에서 시가베팅 비활성(박스권·약세장) 또는 시그널 후보 0개인 경우, 이 루틴을 실행하지 않습니다.

## 1. 실행 타이밍

| 시점 | 행동 |
|---|---|
| **08:00** | 루틴 시작. 시간외 단일가 시세 확인 시작 |
| **08:30** | 시간외 단일가 마감 확인. G5 조건 확정 |
| **08:40~08:50** | 최종 채점 완료 + 진입 전략 확정 |
| **08:55** | 주문 준비 (지정가 OR 시장가 결정) |
| **09:00** | 시초가 진입 실행 |

> ⚠️ 08:50 이후에 새로운 종목을 추가 탐색하지 않습니다. 전일 플래그된 후보 내에서만 최종 결정합니다.

## 2. 입력 데이터

| 항목 | 내용 |
|---|---|
| 레짐 | 강세장(✅) 또는 순환매장(🔄) |
| 시그널 후보 | 최대 3종목 (코드·종가·G1~G3 통과 확인 완료) |

| 토스 호가잔량·체결강도 (장전) | 토스 증권 | 08:50 (프리마켓 데이터) |

## 3. Gate 조건 최종 확인 (G1~G5)

### 전일 종가 분석에서 확인 완료된 Gate (재확인 불필요)

| # | 조건 | 상태 |
|---|---|---|
| G1 | 전일 종가 > 5MA + 5MA 우상향 | ✅ 전일 확인 완료 |
| G2 | 전일 거래량 ≥ 5일 평균 150% | ✅ 전일 확인 완료 |
| G3 | 전일 꽉찬 양봉 (종가 ≥ 고가 × 0.95) | ✅ 전일 확인 완료 |

### 오전에 확인하는 추가 Gate

| # | 조건 | 확인 시점 | 데이터 출처 |
|---|---|---|---|
| G4 | 전일 외국인 OR 기관 순매수 ≥ +5억 | 08:00 | 네이버 (전일 수급 확정) |
| G5 | 시간외 단일가 +1% 이상 **OR** 장전 호재 공시/뉴스 | 08:30 이후 | 네이버 시간외·KIND |

### Gate 판정

```
G1~G5 전부 충족 → 채점 단계 진행
G4 또는 G5 미충족 → 해당 종목 탈락 (시가베팅 대상 제외)
```

> **G5 특이 케이스**: 시간외 단일가가 -0.5%~+1% 사이(애매한 구간)인 경우, 장전 호재 뉴스/공시가 있으면 G5 충족으로 간주합니다. 둘 다 없으면 미충족.

## 4. 채점 조건 (10점 만점)

Gate 통과 종목에 대해 아래 조건을 채점합니다.

### 모멘텀 강도 (×2.0, 최대 4점)

| # | 조건 | 점수 | 데이터 출처 | 확인 시점 |
|---|---|---|---|---|
| M1 | 전일 당일 체결강도 평균 ≥ 110% | 1점 × 2.0 | **토스** | 08:00 (전일 데이터) |
| M2 | 전일 거래량 ≥ 5일 평균 200% | 1점 × 2.0 | 네이버 | 전일 확인 가능 |

### 갭업 신호 (×1.5, 최대 3점)

| # | 조건 | 점수 | 데이터 출처 | 확인 시점 |
|---|---|---|---|---|
| GP1 | 시간외 단일가 +2% 이상 마감 | 1점 × 1.5 | 네이버 | 08:30 이후 |
| GP2 | 장전 호재 뉴스·공시 (실적·수주·정책) | 1점 × 1.5 | 네이버·KIND | 08:00~08:50 |

### 캔들·수급 마감 강도 (×1.0, 최대 3점)

| # | 조건 | 점수 | 데이터 출처 | 확인 시점 |
|---|---|---|---|---|
| C1 | 종가 ≥ 고가 × 0.97 (꽉찬 양봉) | 1점 | 네이버 | 전일 확인 |
| C2 | 외국인 + 기관 동시 순매수 | 1점 | 네이버 | 08:00 (전일 확정) |
| C3 | 장전 매수호가 잔량 : 매도호가 잔량 ≥ 1.2:1 | 1점 | **토스** | 08:50 |

### VKOSPI 점수 보정

| VKOSPI | 보정 |
|---|---|
| < 20 | 보정 없음 |
| 20 ~ 30 | 원점수 × 0.9 |
| > 30 | 시가베팅 비활성 (이 루틴 진입 자체가 차단됨) |

## 5. 갭업 범위 필터

시간외 단일가 또는 프리마켓 예상 시초가를 기준으로 **진입 가능 범위**를 확인합니다.

| 갭업 범위 | 판정 | 행동 |
|---|---|---|
| **0% ~ +3%** | ✅ **적정 범위** | 09:00 시초가 진입 |
| **+3% ~ +4%** | ⚠️ **경계 구간** | 채점 8.0점 이상(A등급 이상)만 진입 허용. 비중 50% 축소. |
| **+4% 초과** | ❌ **진입 보류** | 추격 위험. 진입하지 않음. |
| **-1% 이하** | ❌ **진입 보류** | 모멘텀 실패. 진입하지 않음. |
| **-1% ~ 0%** | ⚠️ **확인 후 판단** | 시초가 첫 1분봉 양봉 전환 시에만 진입 (09:01 확인 후 진입) |

> **핵심 원칙**: 시가베팅은 "적정 갭업 자리에서 모멘텀을 타는 전략"입니다. 과도한 갭업 추격이나 갭다운 진입은 전략의 근본을 훼손합니다.

## 6. 최종 종목 선정

### 선정 기준

1. Gate G1~G5 전부 충족
2. 채점 7.5점 이상 (A등급 이상)
3. 갭업 범위 적정 (0~+3%, 경계 구간은 조건부)
4. 최대 3종목 선정 (점수 순 정렬)

### 등급별 진입 비중

| 등급 | 점수 | 진입 비중 |
|---|---|---|
| S (9.0+) | 강력 | 계좌 배정 한도 100% |
| A (7.5~8.9) | 추천 | 계좌 배정 한도 80% |
| B (6.0~7.4) | — | **시가베팅 진입 불가** (종가 분석 시점에서 관찰만) |

### 복수 종목 진입 시 분배

- 3종목 모두 적격: 균등 분배 (33% × 3)
- 2종목 적격: 1위 60% / 2위 40%
- 1종목만 적격: 해당 종목 100%

## 7. 매도 전략 (분 단위)

### 기본 매도 테이블

| 단계 | 시점 | 조건 | 행동 |
|---|---|---|---|
| 🚪 시초 진입 | 09:00 | 갭업 0~+3% 자리 확인 | 100% 매수 |
| 🔔 1차 익절 | 09:00~09:30 | +2% OR 5분봉 음전환 | 50% 정리 |
| 📈 2차 익절 | 09:30~11:30 | +4% OR 단기 고점 형성 | 30% 추가 정리 |
| ⏰ 마감 정리 | 14:00~14:30 | 시간 도달 (무조건) | 잔량 전량 |
| 🛑 손절 | 즉시 | -2% OR 시가 하향 이탈 | 전량 |

### 레짐별 조정

**강세장**: 2차 익절 목표를 +5%로 상향. 마감 정리 시간을 14:30까지 연장 가능.

**순환매장**: 1차 익절을 +1.5%로 하향(빠른 회수). 11:30 이후 잔량 보유 불가 → 11:30에 무조건 전량 정리.

### 당일 청산 원칙 (철칙)

> **시가베팅은 스윙 전환 대상이 아닙니다.**
>
> - 당일 14:30까지 전량 청산이 원칙입니다.
> - 부득이하게 마감 전 청산하지 못한 경우, **다음 영업일 시초에 무조건 정리**합니다.
> - 어떤 이유로든 "내일 더 오를 것 같아서 보유"하는 판단은 금지합니다.
> - 이 종목이 스윙 가치가 있다면, 당일 종가 분석에서 종가베팅 후보로 재진입을 검토하세요.

## 8. 실행 순서

```
Step 1. 전일 종가 분석 결과 확인 (08:00)
        └─ 시가베팅 레짐 활성 확인 (강세장/순환매장)
        └─ 시그널 후보 리스트 확인 (최대 3종목)
        └─ 레짐이 비활성이면 여기서 종료

Step 2. 해외 시장 + 매크로 체크 (08:00~08:10)
        └─ 미국 S&P500·나스닥 전일 마감 등락률
        └─ 야간선물 KOSPI200 등락률
        └─ 아시아 프리마켓(일본 선물 등) 방향
        └─ 급변 이벤트 체크 (지정학·금리·유가)
        ⚠️ 매크로 리스크 급변 시 (야간선물 -1% 이하 등):
           전 종목 시가베팅 보류 → 종료

Step 3. 추가 Gate 확인 (08:00~08:30)
        └─ G4: 전일 외국인/기관 순매수 ≥ +5억 확인
        └─ G5: 시간외 단일가 시세 확인 (08:30 마감 후)
        └─ G5: 장전 뉴스·공시 확인
        └─ Gate 미충족 종목 탈락

Step 4. 채점 (08:30~08:45)
        └─ M1: 전일 체결강도 (토스 데이터)
        └─ M2: 전일 거래량 배수
        └─ GP1: 시간외 단일가 갭업률
        └─ GP2: 장전 뉴스·공시 유무
        └─ C1~C3: 캔들·수급 강도
        └─ VKOSPI 보정 적용
        └─ 최종 점수 산출

Step 5. 갭업 범위 필터 (08:45~08:50)
        └─ 시간외 단일가 기준 예상 갭업률 산출
        └─ 0~+3%: 진입 / +3~4%: 조건부 / +4% 초과: 보류
        └─ 최종 진입 종목 확정 (최대 3종목)

Step 6. 주문 준비 (08:50~08:55)
        └─ 종목별 진입 비중 결정
        └─ 주문 유형 결정 (시장가 vs 지정가)
        └─ 매도 전략 목표가 사전 계산

Step 7. 진입 실행 (09:00)
        └─ 시초가 확인 → 갭업 범위 내 확인
        └─ 매수 체결

Step 8. 매도 관리 (09:00~14:30)
        └─ 분 단위 매도 테이블 실행
        └─ 14:30 마감 전 전량 청산 확인
```

## 부록: 시가베팅 vs 종가베팅 운용 철학 비교

| 항목 | 시가베팅 | 종가베팅 |
|---|---|---|
| 보유 기간 | 당일(수 시간) | 1~2일 (스윙 전환 시 ~15거래일) |
| 핵심 엣지 | 갭업 모멘텀 지속 | 마감 수급 + 익일 갭업 기대 |
| 진입 시점 | 09:00 시초가 | 14:50~15:30 종가 |
| 손절 기준 | -2% (즉시) | -3~4% (종가 기준) |
| 스윙 전환 | **금지** | V/I/R 게이트로 통제 |
| 레짐 적합도 | 순환매장 > 강세장 | 강세장 > 박스권 |
| 자본 회전 | 매일 100% 회수 | 1~15거래일 묶임 |
| 의사결정 압박 | 높음 (분 단위) | 낮음 (장마감 후 분석) |
