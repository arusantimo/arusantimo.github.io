# Screen Capture Pro 📸

P 키를 누르면 현재 활성화된 웹 화면을 캡처하여 지정된 폴더에 자동으로 다운로드하는 강력하고 직관적인 크롬 확장 프로그램입니다.

## 주요 기능
- **단일 키 단축키**: 페이지 어디서나 `P` (또는 `p`) 키를 누르면 즉시 화면이 캡처됩니다.
- **오작동 방지**: 검색창, 댓글창, 텍스트 에디터 등 글자를 타이핑하는 입력란에서는 단축키가 작동하지 않습니다.
- **저장 위치 설정**: 브라우저 다운로드 폴더(`Downloads`) 기준 상대 경로로 하위 폴더명을 직접 지정할 수 있습니다.
- **두 가지 파일명 형식 지원**:
  1. **날짜 형식**: 다양한 시간 태그(`{yyyy}`, `{MM}`, `{dd}`, `{hh}`, `{mm}`, `{ss}`)를 조합하여 고유한 파일명을 생성합니다.
  2. **자동 증가 형식**: 지정한 파일 접두사(예: `capture_`) 뒤에 숫자가 자동으로 1씩 증가하는 번호 형태의 파일명을 생성합니다. (팝업창에서 직접 번호를 초기화할 수도 있습니다.)
- **미려한 UI**: 글래스모피즘(Glassmorphism) 스타일이 적용된 다크 테마의 설정 팝업과 토글 UI를 제공합니다.
- **성공 알림 피드백**: 캡처가 성공적으로 완료되면 웹페이지 우측 하단에 깔끔하고 모던한 토스트 알림이 표시됩니다.

## 설치 방법 (크롬 로컬 로드)
1. 구글 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 이동합니다.
2. 우측 상단의 **개발자 모드 (Developer mode)** 토글을 켭니다.
3. 좌측 상단에 나타나는 **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
4. 아래 경로의 폴더를 선택합니다:
   `c:\Users\USER\git\arusantimo.github.io\tools\screen-capture`
5. 설치가 완료되면 크롬 툴바의 확장 프로그램 목록에서 **Screen Capture Pro**를 고정(Pin)하여 간편하게 설정할 수 있습니다.

## 설정 팝업 사용법
- **P 키 캡처 단축키**: 단축키 사용 여부를 즉시 토글할 수 있습니다.
- **저장할 하위 폴더명**: `Downloads` 폴더 하위의 경로를 입력합니다. (예: `ScreenCaptures` 입력 시 `Downloads/ScreenCaptures/` 경로에 저장됨)
- **파일명 형식**:
  - **날짜 형식 탭**: 파일명 템플릿에 시간 태그 버튼들을 클릭하여 원하는 포맷으로 조합하여 저장합니다.
  - **자동 증가 형식 탭**: 파일 이름 맨 앞에 붙을 접두사(Prefix)와 시작할 다음 번호를 지정합니다. 번호는 캡처 시마다 1씩 증가하며, 필요시 **번호 초기화** 버튼으로 1로 되돌릴 수 있습니다.
