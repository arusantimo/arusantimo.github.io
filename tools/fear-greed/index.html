데이터 수집 방법 (로컬 Python 스크립트)
scripts/ 폴더의 Python 스크립트를 로컬에서 실행하면 fear_greed_data.json 파일이 생성됩니다. 이 파일을 위의 드롭존에 올리면 대시보드가 업데이트됩니다.
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 크롤링 + 분석 실행
python crawl.py
python analyze.py

# 3. 생성된 JSON을 위 드롭존에 드래그
⚠️ 크롤링 주의사항
  • 🔒 네이버, 디시인사이드는 robots.txt로 크롤링을 제한합니다
  • 👤 스크립트는 개인 로컬 사용을 전제합니다
  • ⏱️ 요청 간 적절한 딜레이(1~3초)가 설정되어 있습니다
  • 🚫 수집 데이터를 무단으로 재배포하지 마세요
