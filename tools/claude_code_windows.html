v2.1.107 2026-04-14
▶ 📋 최근 변경사항 (v2.1.92–107)
  • 긴 작업 중 thinking 힌트를 더 빠르게 표시 (v2.1.107)
  • EnterWorktree 도구에 path 파라미터 추가 — 기존 워크트리로 전환 가능 (v2.1.105)
  • PreCompact 훅 지원 추가 — 컴팩션 차단 가능 (exit 2 또는 {"decision":"block"}) (v2.1.105)
  • 플러그인 monitors manifest 키로 백그라운드 모니터 지원 (v2.1.105)
  • /proactive 가 /loop 의 별칭으로 추가 (v2.1.105)
  • /team-onboarding 명령어 추가 — 팀원 온보딩 가이드 생성 (v2.1.101)
  • OS CA 인증서 스토어 기본 신뢰 추가 — 엔터프라이즈 TLS 프록시 추가 설정 불필요 (v2.1.101)
  • 기본 노력 수준이 API키·Bedrock·Vertex·Enterprise 사용자는 medium에서 high로 변경 (v2.1.94)
  • Amazon Bedrock Mantle 지원 (CLAUDE_CODE_USE_MANTLE=1) (v2.1.94)
  • 대화형 Google Vertex AI 설정 마법사 추가 (v2.1.98)
  • /vim 명령어 제거됨 — Vim 모드는 /config → Editor mode에서 설정 (v2.1.92)
  • /tag 명령어 제거됨 (v2.1.92)
